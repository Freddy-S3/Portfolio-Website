My Books and BlogsBooks and Blogs

Here are some relevant textbooks and blogs I've read over the years.

Clean Code: A Handbook of Agile Software Craftsmanship

Clean Code is divided into three parts. The first describes the principles, patterns, and practices of writing clean code. The second part consists of several case studies of cleaning up code—of transforming a code base that has some problems into one that is sound and efficient. The third part is the payoff: a list of heuristics and “smells” gathered while creating the case studies. The result is a knowledge base that describes the way we think when we write, read, and clean code.

Effective Java

Each chapter consists of several “items” presented in the form of a short, standalone essay that provides specific advice, insight into Java platform subtleties, and updated code examples. The comprehensive descriptions and explanations for each item illuminate what to do, what not to do, and why.

The Clean Coder: A Code of Conduct for Professional Programmers

Robert C. Martin introduces the disciplines, techniques, tools, and practices of true software craftsmanship. This book is packed with practical advice–about everything from estimating and coding to refactoring and testing. It covers much more than than it is about attitude.

Core Java

This book is the definitive guide to writing robust, maintainable code in any version of Java up to V17. The book offers in-depth coverage of fundamental Java programming, including object-oriented programming, generics, collections, lambda expressions, concurrency, and functional programming.

Designing Data-Intensive Applications

This book examines the pros and cons of various technologies for processing and storing data. Difficult issues are explored such as scalability, consistency, reliability, efficiency, and maintainability. In addition, the book explores a variety of tools, including relational databases, NoSQL datastores, stream or batch processors, and message brokers.

AWS Solutions Architect Course

This Learning Path helps you build knowledge on how to design applications and large distributed systems on AWS. This Learning Path presents domain-specific content and includes courses and knowledge checks to prepare you for the AWS Solutions Architect Certification Exam. This path is a guide and presents learning in a structured order.

Fluent Python

This book teaches how to write effective, modern Python 3 code by leveraging its best ideas. The book guides you through idiomatic Python 3 features, Python's core language features, libraries, and teaches you how to make your code shorter, faster, and more readable.

Beyond the Basic Stuff with Python

Sweigart takes you through best practices for setting up your development environment, naming variables, and improving readability, then tackles documentation, organization and performance measurement, as well as object-oriented design. This book also goes over using the command line and other professional tools like code formatters, type checkers, linters, and version control.

Automate the Boring Stuff with Python

In Automate the Boring Stuff with Python, you'll learn how to use Python to write programs that do in minutes what would take you hours to do by hand. Once you've mastered the basics of Python, you'll create programs that effortlessly perform useful and impressive feats of automation
